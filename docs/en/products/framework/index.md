# VSlices Framework

**VSlices Framework** is the .NET implementation product of the VSlices suite.

It provides libraries, primitives, patterns, and development conventions for building domain-oriented software with low ceremony and progressive architecture.

The Framework is heavily influenced by:

- domain-driven design
- vertical slices
- functional programming
- explicit behavior modeling
- compositional design
- and progressive complexity

Its goal is not to hide software engineering concepts behind abstractions, but to make important behaviors, boundaries, dependencies, and domain intent easier to model, compose, and evolve.

## Relationship with VSlices Design

VSlices Framework is intentionally connected to VSlices Design, it not meant to replace design work.

Instead, it exists to help preserve design intent during implementation. Many of the Framework concepts are expected to emerge from real modeling and architectural needs discovered through VSlices Design and validated through real projects.
