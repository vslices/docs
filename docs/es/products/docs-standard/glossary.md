# Glosario de VSlices Docs Standard

Este glosario define términos usados por VSlices Docs Standard.

Docs Standard se enfoca en preservar conocimiento útil mediante estructuras de documentación vivas y coherentes.

## Conceptos de documentación

- __Context document__: un documento que describe el escenario, los actores, las responsabilidades, los límites y el conocimiento compartido relevante para un área.

- __Process document__: un documento que describe cómo se realiza un trabajo, cómo se coordinan las responsabilidades y qué flujo sigue el valor.

- __Use case document__: un documento que describe un comportamiento esperado, su significado de negocio, sus reglas y su validación.

- __Capability document__: un documento que describe una capacidad estable que el negocio o el sistema necesita.

- __Decision record__: un documento que preserva una decisión, el contexto que la motivó, los tradeoffs aceptados y las consecuencias relevantes.

- __Validation note__: un documento que preserva evidencia, hallazgos o aprendizaje obtenido al validar una idea, un comportamiento o una decisión.

- __Support note__: un documento que preserva conocimiento operativo útil para mantener continuidad en el futuro.

## Idea central

Docs Standard trata la documentación como parte del sistema de conocimiento, no como un archivo aislado.

Su objetivo es preservar la intención para que el trabajo futuro siga teniendo contexto.
