# Principios de uso

Docs Standard se usa mejor cuando la documentación ayuda a preservar continuidad.

## Principios

- documentar el conocimiento que el trabajo futuro necesitará
- mantener los documentos conectados con el contexto real
- evitar documentación innecesaria
- preferir claridad sobre volumen

## Idea central

La documentación útil reduce ambigüedad y protege la intención.
