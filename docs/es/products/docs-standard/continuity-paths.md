# Rutas de continuidad

Docs Standard usa rutas de continuidad para mantener el conocimiento conectado entre documentos.

## Qué significa

Una ruta de continuidad ayuda a que una idea descubierta en un documento siga visible en otros documentos relacionados.

Por ejemplo:

- un context document puede alimentar un process document
- un process document puede alimentar un use case document
- un use case document puede alimentar una decision record
- una decision record puede alimentar una validation note

## Idea central

La documentación no debería vivir como piezas aisladas.

Debería formar rutas que permitan seguir el razonamiento a medida que el sistema evoluciona.
