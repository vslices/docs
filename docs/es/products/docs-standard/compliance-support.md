# Soporte para cumplimiento

Docs Standard puede ayudar a los equipos cuando necesitan preservar conocimiento para auditoría, cumplimiento o trazabilidad.

## Enfoque

El valor no está en producir más burocracia.

El valor está en hacer visible el razonamiento, las decisiones y el contexto que respaldan el trabajo.

## Idea central

Si la documentación se usa para cumplimiento, debe seguir siendo útil para el equipo que construye y mantiene el sistema.
