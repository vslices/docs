# VSlices Docs Standard

Docs Standard define estructuras de documentación que preservan la intención del sistema y el razonamiento de ingeniería.

Su objetivo no es producir más documentos. Su objetivo es hacer que la documentación forme parte de la continuidad del trabajo.

Docs Standard ayuda a los equipos a preservar conocimiento sobre:

- lenguaje del dominio
- contextos
- procesos
- casos de uso
- capacidades
- decisiones
- validación
- soporte

## Qué hace Docs Standard

Docs Standard proporciona una estructura para documentar conocimiento de una forma que siga siendo útil con el tiempo.

Ayuda a responder preguntas como:

- ¿Qué conocimiento necesitamos preservar?
- ¿Qué tipo de documento encaja mejor?
- ¿Cómo evitamos que la documentación se desconecte de la implementación?
- ¿Qué debe seguir siendo visible para el trabajo futuro?

## Relación con la suite

Docs Standard funciona junto con VSlices Design, VSlices Method y VSlices Framework.

- Design ayuda a descubrir y modelar el conocimiento.
- Method ayuda a decidir cuándo y cómo documentarlo.
- Framework puede reflejar parte de ese conocimiento en software ejecutable.

Docs Standard existe para sostener continuidad entre pensar, documentar, construir y evolucionar.
