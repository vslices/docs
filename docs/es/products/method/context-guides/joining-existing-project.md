# Incorporarse a un proyecto existente

Esta guía se usa cuando el equipo se incorpora a un proyecto que ya existe.

## Enfoque

Empieza por entender el contexto, las decisiones previas y el conocimiento ya documentado antes de cambiar la estructura.

## Idea central

Un proyecto existente suele contener conocimiento útil mezclado con supuestos antiguos.

El objetivo es recuperar continuidad, no reemplazarlo todo de inmediato.
