# Guías de contexto

Esta sección contiene guías para usar VSlices Method según el tipo de situación en la que se encuentra el equipo.

Las guías no son rutas fijas. Son apoyos para elegir el enfoque inicial más útil.

## Guías incluidas

- Empezar en un escenario no digitalizado
- Empezar en un escenario digitalizado
- Incorporarse a un proyecto existente
- Introducir Method en proyectos en curso

Cada guía parte de una situación distinta.
