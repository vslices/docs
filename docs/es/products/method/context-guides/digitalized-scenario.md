# Empezar en un escenario digitalizado

Esta guía se usa cuando ya existe software, pero el entendimiento sigue siendo incompleto o fragmentado.

## Enfoque

Empieza por observar el sistema actual, sus límites, sus flujos y su comportamiento real antes de proponer cambios.

## Idea central

El software existente puede revelar conocimiento del dominio, pero no debe asumirse que todo el conocimiento ya está correctamente modelado.
