# Bucle de aprendizaje

El bucle de aprendizaje describe cómo la implementación y la validación modifican el entendimiento del equipo.

```text
Understanding -> Documentation -> Design -> Implementation -> Validation -> Learning
```

Ese aprendizaje debe volver a los documentos, a las decisiones y al diseño, para que el siguiente ciclo empiece con mejor contexto.

## Principio

El trabajo entregado no cierra el aprendizaje.

Lo amplía.
