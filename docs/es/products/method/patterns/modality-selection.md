# Selección de modalidad

La selección de modalidad ayuda al equipo a elegir entre Context-First, Problem-First y Slice-First.

La elección depende del tipo de incertidumbre, no de una preferencia abstracta.

## Preguntas útiles

- ¿El dominio todavía es amplio o fragmentado?
- ¿Hay un dolor visible pero la causa no está clara?
- ¿Ya entendemos lo suficiente como para construir una pequeña slice?

## Principio

La mejor modalidad es la que reduce la incertidumbre más relevante para el trabajo actual.
