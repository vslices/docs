# Modelo de colaboración

VSlices Method asume que el trabajo de diseño, documentación e implementación no está aislado.

La colaboración efectiva necesita que las personas compartan contexto, no solo tareas.

## Qué ayuda a preservar

El modelo de colaboración ayuda a preservar:

- understanding
- decisions
- assumptions
- responsibilities
- feedback

## Principio

La colaboración no debería reducirse a coordinar entregas.

Debería ayudar al equipo a mantener visible el razonamiento que sostiene el trabajo.
