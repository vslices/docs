La documentación de VSlices comenzará como un repositorio dedicado a la documentación.

El propósito de este repositorio no es presentar VSlices como una suite terminada, sino definir y documentar los cuatro productos de VSlices mientras se validan mediante Domus Orbis.

La documentación se organizará en torno a:
- Method
- Design
- Docs Standard
- Framework

Pero también incluirá:
- Start Here
- Guides
- notas de Domus Orbis
- decisiones arquitectónicas/de producto

Read the Docs se usará inicialmente para evitar invertir en exceso en branding, identidad visual o diseño del sitio antes de que los límites del producto estén estables.
